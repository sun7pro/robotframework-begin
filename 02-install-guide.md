## Install
Ubuntu, Python 3, Robot Framework

Check python version:
```sh
python3 --version
```

Install `pip` for python 3:
```sh
sudo apt install python3-pip
```

Install robot framework:
```sh
python3 -m pip install robotframework
```

Check robot framework version:
```sh
robot --version
rebot --version
```
